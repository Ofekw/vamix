  • Welcome to the Vamix ReadMe

  • To get started with VAMIX, simply select a media file in the "media" tab

  • You can play the selected media via the player on the top or create new content via the tabs bellow

  • Placing your mouse over a button will show a help dialog at any time inside the application

How to use the media player

To play media, simply browse for a video/audio file or enter the media location in the "Media" Tab. Once you have selected your media, simply use the player control buttons to manipulate playback and volume.
_____________________________________________________________________________

Fullscreen Mode

The media player supports full screen. Full screen controls include: Play/Pause using the SPACE/ENTER key Exit full screen using the ESC key

How to manipulate the audio

There are several audio options, these can be found in the Audio tab. Audio can be extracted or removed from a video and saved as a separate file. Audio can be overlayed/or replaced on a video by selecting an input audio file through the use of the Select Audio button. And Video audio can be replaced completely. All audio can be extract from a video by entering either a duration of 00:00:00 or the media Files length
___________________________________________________________________________________

Adding a Title or Credits Screen

You can add a title screen and closing screen to any video. Enter the text in the corresponding textbox and change the formatting to your liking. Any text you type can be previewed automatically with your current formating settings. Text is limited to letters/numbers and can only be 200 characters long (40 words). Once you are happy with the formatting, select the "apply" button. (note this process take some time). You are able to also customise the background video for the intro/credit screen. There are five options all 5 seconds in length.
___________________________________________________________________________________

Cropping Video

You are able to crop or extract any portion of a media file via the "Video Crop" tab. Simply Input your start time, the duration and press "apply".
__________________________________________________________________________

Save and Load

Saving and Loading Audio and Text can be done via the Project drop down in the menu bar located at the top left. To save, select save from the drop down menu or by using the shortcut Ctrl+S, then enter the save file name. To load, select load from the drop down menu or by using the shortcut Ctrl+), then select the load file.
_______________________

Adding Subtitles

Adding subtitles to any video files is easy. As a video is playing, press the get start/end time buttons (or enter the times manually), this will automatically pull down the current playback times. Insert your text and press "commit". This will commit your subtitle to memory. There is no limit on the number of subtiles simply repeat the process (remember to commit each subtitle). Once you are done, press "Apply Subtitles". Alternatively you can import a .srt file or remove the subtitles for a video.
_______________________________________________________________________________________

Filters

To Apply a filter simply select from the radio buttons and press "apply". Mono - A Black & White filter with extreme black levels and a high grain effect Blur - Applies a guessian blur to the video Box - Adds a red border to the video Flip - Inverts the video along the x/y axis
